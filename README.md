# Par Cristão - Landing Page

Esta é a landing page premium do app Par Cristão.

## Como publicar na Vercel

Existem duas formas fáceis de publicar este projeto na Vercel:

### Opção 1: Upload Direto (Mais Fácil)
1. Acesse [vercel.com/new](https://vercel.com/new)
2. Clique em "Browse" no card "Import a Git Repository" ou arraste a pasta do projeto se tiver a Vercel CLI.
3. **Alternativamente (sem Git):** Instale a Vercel CLI no seu computador:
   - Abra o terminal (CMD ou PowerShell) na pasta do projeto.
   - Digite `npm calculate-size` (se tiver node) ou apenas instale o CLI: `npm i -g vercel`
   - Digite `vercel` e aperte Enter.
   - Siga as instruções na tela (Y para confirmar, Enter para o resto).

### Opção 2: Via GitHub (Recomendado)
1. Crie um repositório no GitHub.
2. Envie os arquivos (`index.html`, `styles.css`, `vercel.json`) para lá.
3. Vá no painel da Vercel e clique em "Add New..." -> "Project".
4. Selecione o repositório do GitHub.
5. Clique em "Deploy".

## Estrutura do Projeto
- `index.html`: Arquivo principal da página.
- `styles.css`: Estilos e animações.
- `vercel.json`: Configuração para a Vercel.
